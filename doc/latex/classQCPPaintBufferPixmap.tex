\doxysubsection{Detailed Description}
A paint buffer based on Q\+Pixmap, using software raster rendering. 

This paint buffer is the default and fall-\/back paint buffer which uses software rendering and Q\+Pixmap as internal buffer. It is used if \mbox{\hyperlink{classQCustomPlot_a7db1adc09016329f3aef7c60da935789}{Q\+Custom\+Plot\+::set\+Open\+Gl}} is false.
